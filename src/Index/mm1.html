Teoría del Método M/M/1

El modelo M/M/1 es uno de los modelos más simples de teoría de colas, utilizado para analizar sistemas de espera en los cuales:

  • M indica que los tiempos de llegada siguen una distribución de Poisson.
  • M indica que el tiempo de servicio sigue una distribución exponencial.
  • 1 significa que hay un solo servidor atendiendo las llegadas.

Este modelo se utiliza comúnmente para sistemas de colas simples, como una línea de espera en un banco o un sistema de atención telefónica con un único operador.

Parámetros del modelo M/M/1

  • λ (Lambda): Tasa de llegada de clientes (número de clientes que llegan por unidad de tiempo).
  • μ (Mu): Tasa de servicio (número de clientes que pueden ser atendidos por unidad de tiempo).
  • ρ (Rho): Factor de utilización, que es la proporción del tiempo que el servidor está ocupado. Se calcula como ρ = λ / μ.

Propiedades del sistema M/M/1

Las propiedades principales del sistema M/M/1 son:

  • Tiempo promedio de espera en la cola (Wq): La fórmula para el tiempo promedio de espera en la cola es Wq = λ / (μ * (μ - λ)).
  • Tiempo promedio en el sistema (W): El tiempo total que un cliente pasa en el sistema se calcula como W = 1 / (μ - λ).
  • Número promedio de clientes en la cola (Lq): El número promedio de clientes esperando en la cola es Lq = λ² / (μ * (μ - λ)).
  • Número promedio de clientes en el sistema (L): El número promedio de clientes en el sistema es L = λ / (μ - λ).

Condiciones para que el sistema sea estable

El sistema M/M/1 es estable si la tasa de llegada es menor que la tasa de servicio, es decir, λ < μ. Si este criterio no se cumple, el sistema estará en un estado de sobrecarga, y los clientes continuarán acumulándose en la cola sin ser atendidos.

Este modelo es ideal para sistemas donde no hay grandes fluctuaciones en las tasas de llegada o servicio, y se utiliza principalmente para entender el comportamiento de sistemas de colas simples.

Instrucciones

Ingresar los datos corespondientes ha la tasa de llegada (λ) y la tasa de servicio (μ), el modelo calcula varias métricas clave del sistema de colas, como:

  • Utilización (p): La fracción del tiempo que el servidor está ocupado.
  • Probabilidad de que no haya clientes en el sistema (P0): La probabilidad de que el sistema esté vacío.
  • Tiempo promedio de espera en la cola (Wq): El tiempo promedio que un cliente espera en la cola antes de ser atendido.
  • Tiempo promedio de espera en el sistema (W): El tiempo total promedio que un cliente pasa en el sistema, incluyendo el tiempo de servicio.
  • Número promedio de clientes en la cola (Lq): El número promedio de clientes que están esperando en la cola.
  • Número promedio de clientes en el sistema (L): El número promedio de clientes en el sistema, incluyendo tanto los que están esperando como los que están siendo atendidos.
